MakeMyTrip Website Redesign

I built a Netflix Clone as my very first project to practice and apply my skills in web development. The main goal of this project was to understand the basics of HTML And CSS while recreating the layout and design of a globally recognized streaming platform. This helped me strengthen my fundamentals and get hands-on experience in structuring web pages, styling components, and adding interactive elements.


The project included key sections such as a hero banner with featured movies, multiple category rows for trending shows and films, and a navigation bar similar to Netflix for smooth browsing. I focused on maintaining a visually clean interface, using images to make the experience more engaging, while keeping the layout responsive for different screen sizes.


Developing this project gave me a strong foundation in front-end development and boosted my confidence to take on more complex projects. It was an important milestone that taught me how to bring a design idea to life through code. The final result is a functional Netflix-inspired website that marked the beginning of my journey as a developer.
