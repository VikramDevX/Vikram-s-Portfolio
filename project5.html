MakeMyTrip Website Redesign

I created the GlamHub Store as my first fully independent project, where I handled everything from planning to execution on my own. The objective was to design and develop a modern e-commerce website with a clean layout, stylish branding, and an easy-to-use interface. This project helped me understand the complete workflow of building a website, from structuring content to ensuring a seamless user experience.


The website features a hero section introducing the store, a well-organized product listing area with clear cards, pricing, and images, along with sections for offers and promotions. I focused on making navigation simple so users could explore products easily, while maintaining an elegant visual design suitable for a fashion and lifestyle brand.


Developing GlamHub Store gave me confidence as a developer because it was the first project I built entirely by myself without external help. It strengthened my skills in HTML And CSS, and taught me how to combine design sense with usability to create a functional product. The end result is a professional, stylish, and user-friendly e-commerce website that reflects my ability to take a project from concept to completion independently.
